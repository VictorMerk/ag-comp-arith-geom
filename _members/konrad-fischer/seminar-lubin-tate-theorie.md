# Lokale Klassenkörpertheorie nach Lubin-Tate

**Course Type:** Seminar  
**Semester:** Summer term 2018  
**Instructors:** Prof. Dr. Böckle, Fischer

## Course Description

Seminar on local class field theory using the Lubin-Tate approach.
