# Klassenkörpertheorie über Funktionenkörpern und Drinfeld Moduln

**Course Type:** Seminar  
**Semester:** Winter term 2014/15  
**Instructors:** Prof. Dr. Böckle, Qiu

## Course Description

Seminar on class field theory over function fields and Drinfeld modules.

- **Type:** Seminar
